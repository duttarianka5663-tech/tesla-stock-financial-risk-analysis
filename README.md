## Project Title
Financial Risk Analysis of Tesla Stock using Python

## Objective
To assess financial risk and analyze price trends 
and volatility of Tesla stock data.

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn

## Key Findings
- Analyzed historical price trends and volatility patterns
- Calculated risk metrics including daily returns and standard deviation
- Identified high-risk periods through statistical analysis

## Dataset
Tesla historical stock data (Yahoo Finance)
